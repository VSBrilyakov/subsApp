# REST API подписок
Небольшой веб-сервер, реализующий REST API для агрегации данных об онлайн подписках пользователей.

# В данном проекте реализовано следующее:
- Работа с GIN для реализации эндпойнтов.
- Работа со структурными тегами.
- Подход Чистой Архитектуры в построении структуры приложения. Техника внедрения зависимости.
- Работа с БД Postgres. Использование sqlx. Запуск из Docker.
- Использование logrus для логирования.
- Генерация файлов миграций(goose).
- Написание SQL запросов.
- Использование swagger для создания API документации.

### Предварительные требования
- Go v1.25.1+
- Docker
- Git

### Установка
```bash
git clone https://github.com/VSBrilyakov/subsApp
cd subsApp
```

## Для запуска приложения:
```bash
make run
```
